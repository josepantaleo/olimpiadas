Guía paso a paso para crear un videojuego en Scratch

Paso 1: Preparar el proyecto y crear variables

Abre Scratch y crea un proyecto nuevo.

Ve a la pestaña Variables y crea las siguientes variables para todos los sprites:

  • Puntuación
  • Vidas
  • Nivel

Inicializa las variables al presionar la bandera verde con este código en el sprite principal (por ejemplo, el jugador):

al presionar bandera verde
fijar [Puntuación v] a (0)
fijar [Vidas v] a (3)
fijar [Nivel v] a (1)

Paso 2: Crear el menú inicial con título, índice, explicación y comandos

  • Crea un fondo nuevo llamado Menú (puedes diseñarlo o usar uno predeterminado).
  • Crea estos sprites:
    • Título: con el nombre de tu juego (usa la herramienta texto).
    • Botón Inicio: un botón para comenzar.
    • Botón Instrucciones: para mostrar la explicación o reglas del juego.

Programa el menú para que se muestre al iniciar, ocultando los sprites del juego:

al presionar bandera verde
mostrar [Título v]
mostrar [Botón Inicio v]
mostrar [Botón Instrucciones v]
// Ocultar sprites del juego (jugador, enemigos, etc)
ocultar [Jugador v]
ocultar [Enemigos v]
ocultar [Objetos v]

Al hacer clic en el Botón Inicio, oculta el menú y comienza el juego:

cuando se clickea [Botón Inicio v]
ocultar [Título v]
ocultar [Botón Inicio v]
ocultar [Botón Instrucciones v]
emitir [Iniciar Juego v]

Al hacer clic en Botón Instrucciones, muestra un sprite con las reglas o instrucciones.

Paso 3: Crear los sprites del juego

Crea al menos 7 sprites, personalizando algunos para dar estilo único:

  • Jugador (edítalo para que tenga tu estilo).
  • Enemigo1 (crea o modifica para que sea diferente).
  • Enemigo2
  • Objeto recolectable (por ejemplo, una moneda).
  • Corazón (para mostrar las vidas o decoración).
  • Botón Inicio (del menú).
  • Botón Instrucciones (del menú).

Paso 4: Crear los escenarios (fondos)

Crea estos fondos:

  • Menú (ya creado en paso 2).
  • Nivel 1.
  • Nivel 2.
  • Nivel 3.
  • Opcional: fondo Game Over o pantalla final.

Para cambiar el fondo, usa:

cambiar fondo a [NombreDelFondo v]

Paso 5: Programar los niveles y lógica de avance

Nivel 1

  • Cambia el fondo al nivel 1.
  • Muestra jugador y enemigos.
  • Programa movimientos del jugador con flechas:
por siempre
  si tecla [flecha derecha] presionada
    cambiar x por (10)
  si tecla [flecha izquierda] presionada
    cambiar x por (-10)
  si tecla [flecha arriba] presionada
    cambiar y por (10)
  si tecla [flecha abajo] presionada
    cambiar y por (-10)
fin
  • Detecta colisiones:
    • Al tocar objetos recolectables, aumentar Puntuación.
    • Al tocar enemigos, perder Vidas.
  • Avanzar a nivel 2 cuando la puntuación sea suficiente:
por siempre
  si <(Puntuación) > 4> y <(Nivel) = 1>
    fijar [Nivel v] a (2)
    emitir [Nivel2 v]
fin

Al recibir el mensaje Nivel2, cambia el fondo y aumenta dificultad (más enemigos, más velocidad).

Repite para nivel 3 con mayor dificultad y puntuación necesaria.

Paso 6: Agregar sonidos

Importa o graba sonidos para:

  • Inicio del juego.
  • Movimientos o saltos.
  • Recolección de objetos.
  • Golpes o daños.

Ejemplo para reproducir sonidos:

cuando presionar bandera verde
reproducir sonido [inicio v] hasta que termine

cuando toca objeto recolectable
reproducir sonido [ding v]

cuando toca enemigo
reproducir sonido [golpe v]

Paso 7: Uso de extensiones

Scratch ofrece extensiones útiles, como:

  • Lápiz: para dibujar en pantalla.
  • Música: para controlar melodías.

Ejemplo con Lápiz para dejar rastro del jugador:

  • Agrega la extensión Lápiz.
  • Código para dibujar mientras el jugador se mueve:
al presionar bandera verde
borrar todo el lápiz

por siempre
  bajar lápiz
  // movimientos del jugador
  si tecla [flecha derecha] presionada
    cambiar x por (10)
  si tecla [flecha izquierda] presionada
    cambiar x por (-10)
  levantar lápiz
fin

Paso 8: Control de vidas y fin del juego

Cuando Vidas llegue a 0, cambia a fondo Game Over y detén el juego.

por siempre
  si <(Vidas) = 0>
    cambiar fondo a [Game Over v]
    detener [todos v]
fin
